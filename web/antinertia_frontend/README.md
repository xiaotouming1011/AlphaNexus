# Antinertia Homepage

Antinertia official landing page (V2) built with React + Vite and a `FloatingLines` animated background.

## Tech Stack

- React
- Vite
- Three.js
- pnpm

## Run Locally

```bash
conda activate antiinertia
pnpm install
pnpm dev
```

Open: `http://127.0.0.1:5173/`

## Build

```bash
pnpm build
pnpm preview
```

## Project Structure

- `src/`: V2 homepage source code
- `src/components/FloatingLines/`: background animation component
- `docs/`: product documentation
- `legacy-v1/`: archived V1 static homepage (`index.html` + `styles.css`)
