# AlphaNexus 多智能体系统评估与迭代框架 v0.1

## 1. 项目概述
AlphaNexus 是一个多智能体投研系统，围绕“信息采集-分析-辩论-风控-结论”构建可追溯决策闭环。

系统链路为：
- 分析员并行分析：市场、社媒、新闻、基本面。
- 投资辩论：Bull / Bear 交叉论证。
- 风控辩论：Aggressive / Neutral / Conservative 三视角评估。
- 最终裁决：输出 Buy / Sell / Hold 交易结论。
- 可观测输出：流程进度、辩论记录、来源引用、结果下载。

其中，`投资风格`当前仅影响新闻筛选层，不影响辩论层与最终结论层。

## 2. 评估体系构建
评估体系分为四层：
- 数据层：数据可用性、完整性、时效性。
- 推理层：分析质量、辩论质量、推理一致性。
- 决策层：结论合理性、风险匹配度、可执行性。
- 体验层：系统稳定性、响应速度、用户可感知质量。

### 2.1 主观评估维度（人工评审）
- 可解释性：结论是否可追溯至明确证据与来源。
- 逻辑一致性：分析、辩论、裁决之间是否一致。
- 可操作性：是否给出可执行的交易动作与风险控制信息。
- 风格匹配度：新闻输出是否与用户投资风格一致。
- 信任感：输出是否具备专业度与可信度。

### 2.2 客观评估维度（系统指标）
- Run Success Rate：任务完整成功率。
- Tool Success Rate：工具调用成功率（含回退后成功）。
- P95 Latency：端到端延迟。
- Evidence Coverage：结论中有证据支撑的观点占比。
- Contradiction Rate：输出内部矛盾率。
- Risk-Profile Match Rate：投资风格与新闻筛选匹配率。
- Badcase Rate：高严重度问题占比。

## 3. 数据评测集构建与更新

### 3.1 评测集获取
- 线上真实请求日志抽样。
- 历史关键行情窗口采样（财报、政策、突发事件等）。
- 不同市场状态采样（上涨、下跌、震荡、事件驱动）。
- 不同行业与市值分层采样。

### 3.2 评测集梳理与结构化
每条评测样本按统一结构管理：
- case_id
- ticker
- trade_date
- selected_module
- risk_profile
- expected_focus
- golden_signals
- hard_fail_rules
- difficulty
- market_regime

评测集分层：
- Core Benchmark：稳定基准集，用于长期横向比较。
- Rolling Benchmark：滚动更新集，用于反映近期市场与产品变化。
- Shadow Set：新能力灰度验证集，不直接影响主榜单。

### 3.3 上线后的持续更新机制
- 日常将线上 badcase 自动进入候选池。
- 周度筛选高价值样本进入 Rolling Benchmark。
- 月度淘汰过时或重复样本，补充新场景样本。
- 版本化管理（v1.0/v1.1/...），支持跨版本对比与回归追踪。

## 4. 线上 Badcase 运营闭环

### 4.1 日常挖掘机制
- 自动采集失败请求、超时请求、人工标注问题样本。
- 按严重级别分级（S0/S1/S2/S3）。
- 建立统一问题标签体系：
  - 数据问题
  - 编排问题
  - 模型问题
  - Prompt 问题
  - 前端展示问题

### 4.2 根因分析
每个 badcase 需产出：
- 现象描述
- 影响范围
- 主因归类
- 复现路径
- 修复动作

### 4.3 迭代推动机制
- 修复动作与责任人绑定（代码、提示词、路由、交互）。
- 修复后样本必须进入回归评测集。
- 次日/次周复盘关键指标变化，验证修复有效性。

## 5. QA 知识库构建

### 5.1 数据来源
- 用户问答日志
- 工单与客服记录
- 运营反馈
- 版本发布说明与变更记录

### 5.2 知识条目标准结构
- question
- intent
- standard_answer
- boundary
- escalation_rule
- last_updated
- owner

### 5.3 维护机制
- 周度增量清洗：去重、合并、规范化。
- 版本发布后全量校对：移除过期答案、更新功能边界。
- 高频问题设定标准答案并进行质量抽检。

### 5.4 质量指标
- QA 命中率
- 一轮解决率（First Response Resolution）
- 转人工率
- 过期答案率

## 6. v0.1 版本边界
本版本用于建立评估与运营的统一框架，不包含：
- 自动打分器实现细节
- 线上平台埋点字段规范
- 报表可视化系统实现
- 自动化回归流水线实现
