# AlphaNexus 评测评分表 v0.1

## 1. 使用说明
- 单次评测对象：1 条 case 的完整运行结果。
- 主观评分区间：1-5 分（1=很差，5=优秀）。
- 客观指标：优先自动采集，无法自动采集时手工记录。

## 2. 主观维度（人工评审）
| 维度 | 定义 | 评分标准（1/3/5） | 分数 |
|---|---|---|---|
| 可解释性 | 结论是否可追溯到具体证据与来源 | 1: 基本不可追溯 / 3: 部分可追溯 / 5: 关键结论均可追溯 |  |
| 逻辑一致性 | 分析-辩论-结论是否一致 | 1: 明显矛盾 / 3: 少量矛盾 / 5: 全链路一致 |  |
| 可操作性 | 是否给出可执行动作与风险约束 | 1: 空泛 / 3: 有方向但不完整 / 5: 可直接执行 |  |
| 风格匹配度 | 新闻筛选是否符合投资风格设定 | 1: 基本不匹配 / 3: 部分匹配 / 5: 高匹配 |  |
| 信任感 | 输出是否专业、可信、无明显幻觉 | 1: 不可信 / 3: 一般 / 5: 高可信 |  |

## 3. 客观指标（系统指标）
| 指标 | 定义 | 目标值（可后续调整） | 实际值 | 是否达标 |
|---|---|---|---|---|
| Run Success Rate | 任务完整成功率 | >= 95% |  |  |
| Tool Success Rate | 工具调用成功率（含回退后） | >= 90% |  |  |
| P95 Latency | 端到端 95 分位耗时 | <= 180s |  |  |
| Evidence Coverage | 有证据支撑观点占比 | >= 80% |  |  |
| Contradiction Rate | 输出内部矛盾率 | <= 5% |  |  |
| Risk-Profile Match Rate | 投资风格与新闻筛选匹配率 | >= 85% |  |  |
| Badcase Rate | 高严重度问题占比 | <= 10% |  |  |
